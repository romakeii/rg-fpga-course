В таблице представлен 3-разрядный код Грея последовательности от 0 до 7

Десятичное значение   Двоичный код Грея
0                     000
1                     001
2                     011
3                     010
4                     110
5                     111
6                     101
7                     100

Разработать 3-разрядный автомат счетчика в коде Грея по модулю 8
У автомата есть вход тактового сигнала и вход синхронного сброса
Нет входа данных, но есть 3-х разрядный выход
После синхронного сброса на выходе должен быть 0
По каждому переднему фронту тактового сигнала счетчик должен переходить к следующему коду Грея
По достижению кода b100 на выходе (последняя строка таблицы), счетчик останавливается

Разработать тестбенч вышеописанного модуля
Тестбенч должен проверять:
- правильность последовательного переключения значений на выходе в соответствии с таблицей
- остановку счетчика при достижении кода b100 на выходе
- правильность работы синхронного сброса
В качестве референсной модели можно использовать функцию преобразования двоичного кода числа в код Грея:
gray = bin ^ (bin >> 1)
